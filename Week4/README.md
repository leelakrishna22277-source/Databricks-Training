# What is PySpark

PySpark is the Python API for Apache Spark.

It is used for:

- Big data processing
- Data analysis
- ETL pipelines
- Machine learning
- Data engineering

PySpark allows us to use Python with Spark.

---

# What is Databricks

Databricks is a cloud platform built on Apache Spark.

It helps to:

- Process huge data
- Run SQL queries
- Build data pipelines
- Perform analytics
- Work with machine learning
