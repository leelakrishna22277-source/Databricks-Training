# Week 1 Day 1 SQL Learning

## Overview

Today I started learning SQL basics and practiced different types of queries using Employee, Department, and Project tables.

## What I Learned

* I learned how to write basic SELECT queries to retrieve data from tables

* I understood how to filter data using WHERE conditions

* I practiced string operations using LIKE and wildcards such as percent and underscore

* I learned how to use functions like LENGTH for string filtering

* I understood how to work with date functions such as YEAR, MONTH, CURRENT_DATE, and DATE_SUB

* I practiced filtering records based on date conditions

* I learned aggregate functions like SUM, AVG, MIN, MAX, and COUNT

* I understood how to group data using GROUP BY



## Mistakes I Faced

* I tried using aggregate functions in WHERE which is not correct
* I used = NULL instead of IS NULL
* I missed adding columns in GROUP BY


## Key Understanding

* WHERE is used before grouping and HAVING is used after grouping
* GROUP BY is important when using aggregate functions
* Writing clean and readable queries is important

## Conclusion

By the end of Day 1, I gained a good understanding of basic SQL concepts and improved my ability to write and debug queries.
